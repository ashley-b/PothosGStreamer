# Pothos Topology Examples

A few example topologies showing GStreamer and Pothos interactions.

## Description

 * 'simple pass through.pothos'
   This example just passes data through GStreamer
 * 'stream penguin radio.pothos'
   This example shows a GStreamer block streaming penguin radio into Pothos audio sink.
 * 'tone to two sinks.pothos'
   This example has a Pothos wave source into a GStreamer block to split the signal into two.
 * 'video source and sink.pothos'
   This example has a GStreamer block to generate a video test pattern into Pothos and displays the test pttern via a GStreamer sink.
